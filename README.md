
# Google Clone Extension


## Features

- Custom Search (webpages and images )
- Extension Pages (devtools, newTab, popup, panel, Option)


## Tech and Tools
 - **Programming language**
   - Typescript
 - **UI**
    - Reactjs
    - TailwindCSS
    - @heroicons/react
 - **Build Tool**: 
    - Vite
    - Websocket (ws)
 - **Code Formatting**
   - Prettier
   - Eslint
   - Commitlint conventional
 - **Test**
   - Jest
 - **CI/CD**
   - Github Actions  
 - **Others**
   - Husky
   - Lint staged
